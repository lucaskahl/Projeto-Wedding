# Wedding website

- HTML
- CSS
  - RESPONSIVE
  - ITCSS
  - RSCSS
  - SASS
  - FLOAT, FLEX and GRID
- JS
  - DOM Manipulate
  - ES6
  - NODE
  - NODE Scripts

## Execute with developer server

- Install the project, unzip it to a folder and with node install the dependencies
-- npm install

- Run the project with:
-- npm run start

## Build css

-- npm run build:css
